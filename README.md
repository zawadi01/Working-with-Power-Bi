# 📊 Power BI Data Analysis & Dashboard Project

## Overview
This project showcases an end-to-end data analysis workflow using Power BI, covering data cleaning, transformation, modeling and the creation of an interactive dashboard.

The goal of the project was to derive meaningful insights from raw data and present them in a clear, visually engaging, and decision-friendly format.

---

## 🎯 Objectives
- Analyze key trends and relationships within the dataset  
- Build a robust data model to support analysis  
- Create an interactive dashboard for data exploration  
- Provide actionable insights based on the data  

---

## 🛠️ Tools & Technologies
- Power BI Desktop  
- Power Query (ETL)  
- DAX (Data Analysis Expressions)  
- Data Modeling (Star Schema)  

---

## 🔄 Data Preparation (ETL)
The dataset underwent several transformation steps:
- Checking and handling missing values  
- Standardized data formats and column names  
- Created calculated columns/ Measures for improved analysis  
- Structured data for efficient querying  

---

## 🧩 Data Modeling
A structured data model was built to optimize performance and usability:
- Implemented a star schema  
- Created fact and dimension tables  
- Established relationships between tables  
- Optimized data for efficient filtering and aggregation  

---

## 📈 Dashboard Features
The interactive dashboard includes:
- Dynamic filters and slicers  
- KPI cards for key metrics  
- Trend analysis visualizations  
- Comparative analysis  
- Drill-down capabilities for deeper insights  

---

## 🔍 Key Insights
Some of the insights derived from the analysis include:
- The relationship between Profits and Sales
- Highest Sale by Product or Category
- Regions and Countries with the highest profit
